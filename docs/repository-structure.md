# Repository Structure

```text
public-repo/
├── assets/brand/   # Maker Express + Hardstack logos and marks
├── resources/      # Public resource catalogs by type
├── funding/        # Grants and funding catalogs
├── mcp/            # MCP server package + docs
├── skills/         # Agent-ready workflow docs
├── schema/         # Canonical schema for resources
├── scripts/        # Validation and maintenance automation
├── tests/          # Automated checks
├── docs/           # Documentation pages
└── README.md
```

## Intent

- Keep `README.md` concise and operational.
- Keep deeper reference material in `docs/`.
- Keep data files in `resources/` and `funding/` source-backed and validator-clean.
